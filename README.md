# MedForecast

Welcome to **MedForecast**, MedForecast is a web-based medical diagnostic platform that integrates Machine Learning and Computer Vision technique to analyse the medical reports to predict the diseases at an early stage and provide the notification personalized recommendation for future health monitoring improvement.

## 🌐 Website
https://github.com/asqar268008/MedForecast/

## Project Overview
The goal of MedForecast is to enable early disease detection by analyzing patient medical data and reports.
The platform provides:
* Disease prediction using ML models.
* Insights from computer vision analysis on medical scans/reports.
* Personalized health monitoring recommendations.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Django
- **Database:** Postgresql, MongoDB

## Getting Started
1. **Clone the Repository:**

```
git clone "https://github.com/asqar268008/MedForecast"
cd MedForecast
```

2. **Install Dependencies:**

 run `pip insatll -r requirements`.
run `npm install`.

3. **Setup MongoDB:**

- Create a `.env` file in the `Server` folder.
- Add your MongoDB URI and POSTGRESQL_PASSWORD and Database name to the `.env` file.
  ```
  MONGODB_URI=<your-mongodb-uri>/database-name
  POSTGRESQL_PASSWORD=<your-postgresql password>
  ```
4. **Start the Application:**

- To start the frontend, run `npm start`.
- To start the backend, run `python manage.py runserver`.

5. **Access the Application:**

- Open your browser and visit `http://localhost:3000` to access the application.
